# DigiFatura

> Gestão de faturas com OCR, estatísticas e notificações em tempo real.

---

## Sobre o Projeto

**DigiFatura** é uma aplicação web de gestão de faturas que simplifica o processo de captura, organização e análise de documentos financeiros.

Através de **OCR (Reconhecimento Ótico de Caracteres)**, a aplicação permite capturar automaticamente as informações das faturas através do dispositivo móvel, eliminando por completo a necessidade de introdução manual de dados.

O objetivo central foi criar uma ferramenta que tornasse a gestão financeira pessoal e empresarial mais **eficiente, acessível e inteligente** — centralizando tudo num único dashboard.

---

## Funcionalidades

- 📷 **Captura via OCR** — Fotografa a fatura com o telemóvel e os dados são extraídos e preenchidos automaticamente.
- 🔄 **Sincronização em Tempo Real** — Todas as alterações refletem instantaneamente em todos os dispositivos ligados à conta.
- 📊 **Dashboard de Estatísticas** — Visualização de despesas por categoria, evolução mensal e análise de padrões de consumo.
- 🔔 **Sistema de Notificações** — Alertas automáticos para faturas pendentes, vencimentos e resumos periódicos.
- 🗂️ **Organização por Categorias** — Classificação automática e manual, com exportação de dados em múltiplos formatos.
